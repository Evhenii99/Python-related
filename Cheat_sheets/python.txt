Дії які можна робити:

+ додавання
- віднімання
* - множення
/ - ділення
// - ділення з двух цисел не тільки з цілим число ^ print(7 / 2)  # 3.5
** - степінь тіпа ^ print(6 ** 2)  # Возводим число 6 в степень 2. Результат - 36
та багато інших(гуглимо)

Основні вбудовані типи в пайтон

None – невизначене значення змінної.
Boolean Type – логічний тип.
Numeric Type – числа:
int – ціле число;
float – число з плаваючою точкою (дійсне число);
complex – комплексне число.
Sequence Type – послідовності:
list – список;
tuple – кортеж;
range – діапазон.
Text Sequence Type – рядки;
str – рядки.
Binary Sequence Types – бінарні послідовності:
bytes – байти;
bytearray – масиви байт 24;
memoryview – спеціальні об’єкти для доступу до внутрішніх даних об’єкта через protocol buffer.
Set Types – множини:
set – множина;
frozenset – незмінювана множина.
Mapping Types – словники:
dict – словник.


Синтаксис
[] - викоритовуються для створення списку, тобто все що всередині квадратних дужок буде розпізнаватися як послідовний список. Всередині списку елементи розділяються комами.
() - круглі дужки використовуються в функціях
{} - дужки для словника
